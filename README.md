# Xuan
Xuan是一款开源的围棋AI，其名称来源于中国古代围棋名著 **《玄玄棋经》**。
本项目是Xuan的衍生子项目,为国际象棋开发.

# 技术架构|Architecture
引擎:Python+torch+sqlite3

前端:Electron+Vue+Node.js

# 开发流程
可能需要的软件:Pycharm,VS Code,SourceTree

# 安装与运行
## 前端(/front)
```
npm install
npm run dev
```
## 引擎(/engine)

```
python main.py
```
